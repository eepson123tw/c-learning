# 格式化輸出指令符

## 下面是一個表格，列出了 C 語言中一些常見的格式化輸出指令符及其用途：

| 格式指令符 | 描述                                       |
| ---------- | ------------------------------------------ |
| `%d`       | 用於十進制整數的輸出。                     |
| `%i`       | 與`%d`相似，用於十進制整數的輸出。         |
| `%u`       | 用於無符號十進制整數的輸出。               |
| `%f`       | 用於浮點數的輸出。                         |
| `%lf`      | 用於雙精度浮點數的輸出。                   |
| `%e`       | 用於科學計數法表示的浮點數的輸出。         |
| `%g`       | 用於浮點數的輸出，`%f`和`%e`中較短的一種。 |
| `%x`       | 用於十六進制整數的輸出（小寫字母）。       |
| `%X`       | 用於十六進制整數的輸出（大寫字母）。       |
| `%o`       | 用於八進制整數的輸出。                     |
| `%s`       | 用於字符串的輸出。                         |
| `%c`       | 用於單個字符的輸出。                       |
| `%p`       | 用於指針的地址輸出（以十六進制表示）。     |
| `%%`       | 用於輸出百分比符號`%`。                    |

這些格式指令符在 C 語言中廣泛使用，幫助程序員控制輸出的格式，使輸出更加可讀和有序。了解並熟練運用這些格式指令符對於 C 語言的學習和使用非常重要。
